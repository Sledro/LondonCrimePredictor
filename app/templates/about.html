The Project

Welcome to the London Crime Predictor. This project was created as part of my final year project for our Data science module on our Software Development BSc Honors course.

We were assigned a task to create a Data Science related project using a large dataset (> 30,000 rows). I decided to analyse crime reports in greater London Metropolitan area and try to make predictions based on that analysis.

The Dataset

This dataset provides a complete snapshot of crime, outcome, and stop and search data, as held by the Home Office from late 2014 through mid 2017 for London, both the greater metro and the city..

Data Source:https://www.kaggle.com/sohier/london-police-records
